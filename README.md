# Staybnb Frontend Project

An Airbnb-inspired frontend website built using HTML, CSS, and JavaScript.

## Features

- Responsive design
- Multi-page layout
- Property listing interface
- House details page
- Navigation components
- Modern UI inspired by Airbnb
- Mobile-friendly design

## Tech Stack

- HTML5
- CSS3
- JavaScript
